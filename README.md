# ZoomPreviewPicture
仿 https://github.com/yangchaojiang/ZoomPreviewPicture
解决全屏查看问题--状态栏导致的

 ### 特点
   * 1.支持自定义图片加载框架。
   * 2.支持重写activity,完成切换切换效果。
   * 3.图片查看 缩放 拖拽下拉缩小退出。
   * 4.支持自定义activity,Fragment。
   * 5.支持视频和自定义视频播放控件。 
   * 6.支持类似微信朋友圈照片九宫格和微信聊天图片预览。
   * 7.指示器类型选择 [圆点模式(贝塞尔圆点指示器)](https://github.com/yanyiqun001/bannerDot)和数字模式。
   * 8.增加接口实体类。不在使用数据转化。
   * 8.支持GIF显示。
